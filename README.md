# AWS Resource Listing Shell Script

A simple Bash shell script to list different AWS resources using the AWS CLI.  
This project helps in understanding **AWS CLI**, **Linux shell scripting**, and **basic automation**.

---

##  Features

This script can list the following AWS resources:

- EC2 Instances  
- S3 Buckets  
- RDS Instances  
- DynamoDB Tables  
- Lambda Functions  
- VPCs  
- EBS Volumes  
- IAM Users  
- CloudFront Distributions  
- CloudWatch Alarms  
- CloudFormation Stacks
